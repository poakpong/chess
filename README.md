# ♟️ Chess Online

เกมหมากรุกออนไลน์สำหรับเล่น 2 คน คนละที่ สร้างด้วย HTML/CSS/JavaScript + WebRTC

## 🎮 วิธีเล่น

### สร้างห้องใหม่
1. เปิดเว็บไซต์
2. กด "สร้างห้องใหม่"
3. ส่งลิงก์ให้เพื่อน
4. รอเพื่อนเข้ามา แล้วเริ่มเล่น!

### เข้าร่วมห้อง
1. ใส่รหัสห้อง (เช่น `ABC123`)
2. กด "เข้าร่วมห้อง"
3. เริ่มเล่น!

## ✨ ฟีเจอร์

- ✅ หมากรุกคลาสสิก (กติกาครบถ้วน)
- ✅ เล่น 2 คน คนละที่ (WebRTC P2P)
- ✅ สร้างห้องได้ ป้องกันคนเข้าเกิน 2 คน
- ✅ จับเวลา 10 นาทีต่อฝ่าย
- ✅ ประวัติการเดิน
- ✅ แสดงหมากที่ถูกกิน
- ✅ รองรับการเลื่อนขั้น (Promotion)
- ✅ การเข้าปราการ (Castling)
- ✅ การกินผ่านทาง (En Passant)
- ✅ ระบบยอมแพ้ / เสนอเสมอ

## 🛠️ เทคโนโลยี

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Network:** PeerJS (WebRTC)
- **Deploy:** GitHub Pages

## 🚀 Deploy

เปิดใช้งาน GitHub Pages:
1. ไปที่ Settings → Pages
2. Source: Deploy from a branch
3. Branch: `main` → `/ (root)`
4. รอ 2-3 นาที

**URL:** `https://poakpong.github.io/chess`
